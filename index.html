Data Localization Suite (DLS)

Cloudflare DLS is a set of products that helps customers who want to maintain local control over their traffic while retaining the security benefits of a global network.

Regional Services

Regional Services enables Cloudflare to decrypt and inspect HTTPS traffic only within the customer's selected region, ensuring security by decrypting and applying Layer 7 security measures at the designated regional data center rather than the initial global entry point.

Regional Services is configured on Zone hostname-level.

Customer Metadata Boundary

Customer Metadata Boundary ensures that end user traffic logs (Customer Logs) processed by Cloudflare stay within the customer's chosen region.

Customer Metadata Boundary is configured on Account-level.

  • httpRequestsAdaptiveGroups

    Earth üåç

    Without Customer Metadata Boundary, essentially set to Earth, all Customer Logs, like i.e. HTTP Requests, are available.
