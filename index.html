Data Localization Suite (DLS) Demo

Cloudflare DLS is a set of products that helps customers who want to maintain local control over their traffic while retaining the security benefits of a global network.

Cloudflare Workers

Without Regional Services, essentially set to Earth, this Worker will always return a "colo" that is closest to the client. 

Loading...

Cloudflare Workers + Regional Services

With Regional Services set to United States of America (US), this Worker will always return a "colo" within the US. 

$ curl -s https://workers.dlsdemo.com/ | grep -o '"colo": *"[^"]*"'

"colo": "IAD"

Cloudflare Pages + Regional Services

$ curl -sI https://landing.dlsdemo.com/ | grep -oE 'cf-ray: [^[:space:]]+' | grep -oE '[^-]+$' | awk '{print "\"colo\":\""$0"\""}'

"colo":"IAD"
